# Competitive Programming Solutions
This repository contains my solutions to the various programming problems I have solved, primarily for my university's competitive programming team. As of January 2026, our team is primarily using `C++` for solutions and working through the `USACO Guide`. Before this, we primarily used `Python` or `Java` and sourced problems from `Kattis`.

## How the Repo is Organized
The repo is organized by general topic names as described according to the USACO Guide. For problems that do not come from the USACO Guide, do not fit a specific topic, or have not been sorted (mostly from before January 2026), I have created an `Unsorted Problems` folder.

Under each topic folder, there are individual folders for each problem its solution(s).
